    1. 常用标签

    Html：超文本标记语言

    <!DOCTYPE html>
<html>
<head>
<meta charset="utf-8">
<title>这是页面的标题</title>
</head>
<body>
    <h1>我的第一个标题</h1>
    <p>我的第一个段落。</p>
</body>
</html>

    属性：属性是 HTML 元素提供的附加信息。

    HTML 链接由 <a> 标签定义。链接的地址在 href 属性中指定：
<a href="http://www.runoob.com">这是一个链接</a>

    字体标签： 规定文本的字体，字体尺寸，字体颜色

    顺便说一下，font 在Html4中，不赞成使用。在Html5中，不支持使用。

    <font>我是字体标签</font>
<font size="1">我修改字体大小</font> 
<font face="黑体">我修改字体样式</font>
<font color="red">我修改字体颜色</font>

    段落标签：定义一个段落

    <p>这是一个段落 </p>
<p>这是另一个段落</p>

    标题标签：标题（Heading）是通过

    -

    标签进行定义的。

    定义最大的标题。

    定义最小的标题。

    <h1>这是一个标题。</h1>
<h2>这是一个标题。</h2>
<h3>这是一个标题。</h3>

    文本格式化：HTML 使用标签 (“bold”) 与 (“italic”) 对输出的文本进行格式, 这些HTML标签被称为格式化标签

    <b>加粗文本</b><br><br>
<i>斜体文本</i><br><br>
<code>电脑自动输出</code><br><br> 
<sub> 下标</sub> 和 <sup> 上标</sup>

    img标签：在浏览器中显示图片

    <img border="0" src="/images/pulpit.jpg" 
alt="Pulpit rock" width="304" height="228">

    超链接：HTML 使用超级链接与网络上的另一个文档相连。几乎可以在所有的网页中找到链接。点击链接可以从一张页面跳转到另一张页面。使用 target 属性，你可以定义被链接的文档在何处显示。

    <p><a href="/index.html">本文本</a> 是一个指向本网站中的一个页面的链接。</p>
<p><a href="//www.microsoft.com/">本文本</a> 是一个指向万维网上的页面的链接。</p>

    2. 表格设计

    表格：表格由 标签来定义。每个表格均有若干行（由 标签定义），每行被分割为若干单元格（由

    标签定义）。字母 td 指表格数据（table data），即数据单元格的内容。数据单元格可以包含文本、图片、列表、段落、表单、水平线、表格等等。其中border表示的是边框，如果不定义边框属性，表格将不显示边框。有时这很有用，但是大多数时候，我们希望显示边框。width 表示的是宽度，bgcolor表示的是背景颜色，align表示在文档中的对齐方式。valign三种垂直对齐方式。

    使用边框属性来显示一个带有边框的表格：

    <table border="1">
    <tr>
        <td>row 1, cell 1</td>
        <td>row 1, cell 2</td>
    </tr>
    <tr>
        <td>row 2, cell 1</td>
        <td>row 2, cell 2</td>
    </tr>
</table>
